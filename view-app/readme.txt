This app shows data fetched from 'https://api.publicapis.org/entries' into two different viewing types, table view and list view.

Steps to run -

1. Open node command prompt.
2. cd to view-app folder.
3. Run following command to install required dependencies.
   npm install
4. Run following command to start the client application.
   npm start
5. Once started, a webpage will open or visit localhost:3000. Toggle the switch button to switch different viewing styles.
